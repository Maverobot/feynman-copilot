# Feynman Plugin

Research-first AI skills and agents for GitHub Copilot CLI.

## Skills (18)

| Skill | Description |
|-------|-------------|
| deep-research | Run thorough, source-heavy investigations with cited output |
| literature-review | Lit review with paper search and primary-source synthesis |
| paper-writing | Turn research findings into paper-style drafts |
| peer-review | Simulated tough but constructive AI research review |
| paper-code-audit | Compare paper claims against public codebase |
| source-comparison | Compare multiple sources with grounded matrix |
| replication | Plan or execute replication of papers/claims |
| alpha-research | Search, read, and query academic papers via alpha CLI |
| eli5 | Explain research in plain English |
| autoresearch | Autonomous experiment optimization loop |
| docker | Execute research code in isolated containers |
| modal-compute | Serverless GPU workloads via Modal |
| runpod-compute | Persistent GPU pods via RunPod |
| watch | Recurring research monitoring |
| preview | Render and view Markdown/LaTeX/PDF artifacts |
| session-log | Capture work, findings, and next steps |
| session-search | Search past session transcripts |
| jobs | Inspect background research processes |

## Agents (5)

| Agent | Description |
|-------|-------------|
| feynman | Lead research agent — plans, delegates, synthesizes, delivers |
| researcher | Evidence-gathering subagent with strict integrity rules |
| reviewer | Skeptical but fair peer reviewer with inline annotations |
| writer | Turns research notes into structured briefs and drafts |
| verifier | Post-processes drafts with inline citations and URL verification |
