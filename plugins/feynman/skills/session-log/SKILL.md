# Session Log

Write a session log for the current research work.

## What to Capture

- **What was done** — summarize work completed in this session
- **Strongest findings** — key decisions, discoveries, or results
- **Open questions** — unresolved risks and uncertainties
- **Next steps** — concrete actions for the next session
- **Artifacts** — reference any files written to `notes/`, `outputs/`, `experiments/`, or `papers/`
- **Sources** — include direct URLs for any external claims that matter

## Output

Save the log to `notes/` as markdown with a date-oriented filename:

```
notes/YYYY-MM-DD-<brief-description>.md
```

Example: `notes/2025-01-15-scaling-laws-review.md`
